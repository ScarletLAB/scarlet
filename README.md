### Options
***You can add options before loadstring to configure Rayfield theme & keybinds.***
```lua
AWTOptions = {
    Theme = "Minty",--Default,AmberGlow,Amethyst,Bloom,DarkBlue,Green,Light,Ocean,Serenity
    Keybind = "K"
}
```
