### Options
***You can add options before loadstring to configure Rayfield theme & keybinds.***
```lua
Options = {
    Theme = "Minty", -- Default, Minty, Skeet, AmberGlow, Amethyst, Bloom, DarkBlue, Green, Light, Ocean, Serenity
    Keybind = "K"
}
```
